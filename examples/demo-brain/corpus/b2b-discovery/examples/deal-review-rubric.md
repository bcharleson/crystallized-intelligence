# Deal Review Rubric

Use weekly for every open opportunity above stage 0.

| Question | Green | Yellow | Red |
|----------|-------|--------|-----|
| Pain in prospect's words? | Yes, documented | Vague | None |
| Champion identified? | Named, engaged | Title only | Missing |
| Economic buyer path? | Clear | Assumed | Unknown |
| Timeline | Dated event | Soft quarter | None |
| Next step | Scheduled | Tentative | "They will reach out" |

**Actions:**

- All green → advance stage with mutual plan
- Any red → disqualify or nurture with calendar date; no demo push
- Yellow only → one targeted discovery before more outreach

Reject tier-4 advice to "keep everything open." Pipeline quality beats volume.
