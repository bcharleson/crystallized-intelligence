# Demo Brain

This is a public-safe sample brain used to demonstrate the framework.

It is intentionally small and synthetic. Its job is to show:

- a valid brain manifest
- one complete domain
- crystallized outputs
- a tiny knowledge/source/example footprint

The demo domain is `specialty-coffee`, chosen because it can show heuristics,
trade-offs, and layered knowledge without exposing any proprietary business
method.
